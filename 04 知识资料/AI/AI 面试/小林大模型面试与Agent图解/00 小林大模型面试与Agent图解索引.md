# 小林大模型面试与Agent图解索引

## 来源范围

本专题由 `01 未整理笔记` 中的《图解 Agent》《大模型面试题》以及相关小林面试笔记链接解析而来。共解析 AI 相关链接 89 个，覆盖 Agent、RAG、工具调用、LLM 工程和 Agent 工程方法论。

## 学习路径

1. [[01 Agent 核心面试体系]]：先掌握 Agent 定义、架构、Workflow/Tools、ReAct、记忆和多 Agent。
2. [[02 RAG 工程面试体系]]：再掌握 RAG 从文档入库到检索生成、评估和持续更新的闭环。
3. [[03 工具调用 MCP Skill 与协议]]：补齐 Function Calling、MCP、Skill、A2A 和通信协议。
4. [[04 LLM 工程面试体系]]：理解 Transformer、训练微调、推理优化、部署和评测。
5. [[05 Agent 图解专题与工程方法论]]：吸收长文图解中的 OpenClaw、GraphRAG、Harness Engineering、Loop Engineering。
6. [[06 逐链接解析索引]]：查看每个链接的解析归属和来源。

## 核心地图

- Agent 层：把 LLM、工具、记忆、规划和执行循环组织成能完成目标的系统。
- RAG 层：把外部知识接入模型，核心难点在切分、召回、重排、上下文构造、引用和评估。
- 工具层：Function Calling 是模型输出工具调用意图，MCP 是工具上下文协议，Skill 是可复用任务知识包。
- 模型层：LLM 工程关注 Attention、位置编码、训练、微调、缓存、量化、部署和模型选型。
- 工程方法层：Harness/Loop Engineering 强调可反馈、可复用、可迭代的 Agent 工作环境。
