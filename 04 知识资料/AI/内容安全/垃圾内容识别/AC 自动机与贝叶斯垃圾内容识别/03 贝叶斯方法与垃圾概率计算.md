# 贝叶斯方法与垃圾概率计算

## 1. 为什么不能只用 AC 自动机

AC 自动机能快速找出关键词，但关键词命中不等于内容一定违规。

例如某些词在垃圾内容中常见，但正常讨论中也可能出现。如果直接拦截，就会造成误伤。因此需要一个能衡量“这个词对垃圾/正常内容的区分度”的方法。

## 2. 贝叶斯方法的作用

贝叶斯思想用于根据样本统计估计：当一条内容出现某个词时，它属于垃圾内容的概率有多高。

原文用：

- `S` 表示垃圾评论；
- `H` 表示正常评论；
- `W` 表示某个词。

核心公式：

```text
P(S|W) = P(W|S)P(S) / [P(W|S)P(S) + P(W|H)P(H)]
```

在没有更多先验知识时，可假设：

```text
P(S) = P(H) = 0.5
```

## 3. 原文示例

假设某个词 `sb`：

- 在 1000 条垃圾评论中，500 条包含该词，则 `P(W|S)=0.5`；
- 在 1000 条正常评论中，2 条包含该词，则 `P(W|H)=0.002`。

代入公式可得：

```text
P(S|W) ≈ 0.996
```

说明该词对识别垃圾评论有很强区分度。

## 4. 正向词也有价值

贝叶斯方法不仅能找“垃圾倾向词”，也能找“正常倾向词”。

例如某个词在正常评论中频繁出现，而在垃圾评论中很少出现，则它可以作为降低拦截概率的信号。

## 5. 多词联合与结构关系

原文还提到可以进一步考虑：

- 多个词语联合共现；
- 词与词之间的空间结构；
- 主关键词与可选词之间的组合；
- 类别内不同策略的组合判定。

这说明该系统并不是朴素地“命中某个词就拦截”，而是在策略层支持组合规则和统计权重。

## 6. 贝叶斯在系统中的定位

```text
AC 自动机：负责快、全、可配置的召回
贝叶斯方法：负责从样本中沉淀高区分度策略
策略系统：负责把统计结果变成可上线规则
```

它更像是帮助运营和算法人员“发现策略”的工具，而不仅是一个最终分类器。
