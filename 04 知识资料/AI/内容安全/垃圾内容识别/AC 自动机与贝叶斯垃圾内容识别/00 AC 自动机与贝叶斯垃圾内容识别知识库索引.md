## 核心地图

```text
垃圾内容识别
  ├─ 业务问题：违法、广告、辱骂、色情、赌博等内容影响社区质量
  ├─ 旧方案问题
  │   ├─ 纯文本分类：泛化有限，召回不足
  │   ├─ DFA/关键词：召回强但误伤高
  │   └─ 人工审核：延迟高、人力成本大
  └─ 新方案：AC 自动机 + 贝叶斯策略
      ├─ AC 自动机：多关键词高效匹配，圈定类别
      ├─ 贝叶斯：根据词在垃圾/正常样本中的频率计算倾向
      ├─ 策略层：主关键词 + 可有可无词 + 组合关系
      └─ 运营闭环：样本分析 -> 策略制定 -> 离线验证 -> 上线
```
