# Vibe Coding 从0到1保姆级教程

## 一句话总结

不需要写一行代码——20 步把想法变成真实上线产品：买 Coding Plan → 起名+域名+服务器+备案 → Agent 写代码 → GitHub 版本管理 → 本地验收 → 洁癖 Skill 整理文档 → 搭测试流程 → 部署上线 → 迭代循环。

---

## 20 步全流程

### 准备阶段（1-6）

| 步骤 | 做什么 | 关键提醒 |
|------|--------|---------|
| **1** | 买一个国产大模型 Coding Plan | Kimi K3/Qwen 3.8/DeepSeek V4，差距没想象大 |
| **2** | 下载各家官方 Agent 编程产品 | ZCode/Qoder/Kimi Code，买哪家就用哪家 |
| **3** | 想好产品名字 | 名字绑定 logo/域名/备案/小程序/App Store，改起来成本大一百倍。顺便搜一下有没有同名的 |
| **4** | 立刻注册域名 | 火山引擎/腾讯云/阿里云。**不要等**——之后想买可能买不起了 |
| **5** | 买一台服务器 | 腾讯云轻量新客 99-199/年。让 Agent 帮你选配置。轻量产品也考虑 Serverless/云开发 |
| **6** | 同步做 ICP 备案 | 国内必须备案，不然域名被运营商拦截。审核 1-2 周，不要等产品做完才想起来 |

### 开发阶段（7-13）

| 步骤 | 做什么 |
|------|--------|
| **7** | 新建项目文件夹，跟产品同名 |
| **8** | 把 Agent 工作目录指向这个文件夹，开始开发 |
| **9** | 注册 GitHub，让 Agent 创建**私有仓库**，提交初始版本 |
| **10** | 开启 **Plan 模式**，用人话描述你最核心的功能。不需要画原型图、不需要 PRD——先看到正反馈 |
| **11** | 让 Agent 按 Plan 执行开发 |
| **12** | **本地验收**：`localhost:3001` 打开看效果，不满意就让 AI 改 |
| **13** | 跑**洁癖 Skill**——专门整理项目文档/代码规则/AI 记忆，消除矛盾，确保新对话能快速恢复上下文 |

> 洁癖 Skill 开源地址：https://github.com/KKKKhazix/khazix-skills/tree/main/neat-freak

### 上线阶段（14-18）

| 步骤 | 做什么 |
|------|--------|
| **14** | 跟 Agent 学专业名词：Git、分支、main、PR、CI、测试流程。不需要全懂，但要能听懂 |
| **15** | 搭测试流程。推荐 GitHub Pro（$4/月）——**分支保护**：PR 必须过 CI 测试才能合，防止手抖推坏代码到主分支 |
| **16** | 部署到服务器：告诉 Agent 服务器 IP，它会处理一切 |
| **17** | 域名连项目：Agent 配 DNS + HTTPS + 免费证书 |
| **18** | **迭代循环**：新建分支 → 说需求 → Agent 开发 → 洁癖 Skill 整理 → PR 过 CI → 合并 → 部署 |

## 两个最重要的原则

### 19. 可以不懂代码，但必须了解系统架构

> AI 是你的执行者，但你必须知道逻辑怎么走。

- 每个页面背后对应哪个接口
- 每个接口做了什么
- 数据库有哪些表、哪些字段、怎么被用到的
- 用户点按钮后数据怎么流转

**不要让渡自己的思考**——AI 解释方案，你听懂、同意、或有更好方案，才让它动手。

### 20. 一定要学运维

用户少时无所谓，用户多起来后：
- 缓存机制、防爬虫、防 DDoS
- 流量带宽费用控制——不要被人一天刷 4000 块才补课

## 补充：国内用户——CloudBase 一条龙（大幅简化）

原教程 20 步对国内场景其实偏重了。如果你只想**快速把想法变成产品**，不需要自己买服务器、配域名、搞备案的骚操作，直接上腾讯云 CloudBase：

| 原教程步骤 | CloudBase 替代 |
|-----------|---------------|
| 买服务器 | ❌ 不需要——CloudBase 云函数/云托管自带计算 |
| 配域名+备案 | ❌ 不需要——静态托管自带 `.tcloudbaseapp.com` 域名 |
| 装数据库 | ❌ 不需要——NoSQL 数据库开箱即用 |
| 配 Nginx/HTTPS | ❌ 不需要——平台自动处理 |

**三步搞定**：

1. 注册 CloudBase，创建一个环境（免费版够用）
2. 装 CloudBase MCP → Agent 直接操作数据库/存储/云函数
3. 跟 Agent 说「帮我做一个 XXX」→ 它会建表、写代码、部署一条龙

已创建 `cloudbase` Skill（`hermes skills list` 可查），用到时 `/skill cloudbase` 加载即可。微信小程序/公众号类产品的最佳选择——天然集成微信登录和微信支付。
