# Agnes 图像与视频模型实测

## 1. 图像模型：Agnes-Image-2.1-Flash

原文强调 Agnes 图像模型不只是生成插画，也能生成架构图、流程图等程序员常用文档素材。

官方图像模型教程：<https://agnes-ai.com/doc/agnes-image-21-flash>

## 2. 图像能力测试一：架构图 / 流程图

### 测试场景

生成一张微服务架构图，要求：

- 横版 16:9；
- 顶部用户和移动 App；
- 中间 API Gateway；
- 下方用户服务、订单服务、库存服务；
- 底部 MySQL、Redis、Kafka；
- 圆角矩形；
- 英文服务名；
- 主路径粗线，异步路径虚线；
- 蓝灰配色，不要花哨。

### 原文观察

生成结果可以直接放入技术评审 PPT：

- 组件层次明确；
- 连线带箭头；
- 主路径和异步路径区分明显；
- 蓝灰圆角矩形风格干净。

### 可复用 Prompt 模板

```text
请生成一张清晰的 [系统名称] 架构图，横版 16:9。

要求：
- 顶部是 [入口层]
- 中间是 [网关/核心入口]
- 下方分出 [核心服务列表]
- 底部是 [基础设施列表]
- 每个组件用圆角矩形，标注英文服务名
- 连线带箭头，主路径用粗线，异步路径用虚线
- 整体配色干净，蓝灰为主，不要花哨
- 不要文字水印，不要装饰图形
```

## 3. 图像能力测试二：创意图片

原文测试了两个中文创意场景：

1. 大学生秋招拿到腾讯 offer 的真实摄影风图片；
2. 高考金榜题名的中式国潮插画。

作者认为：

- 真实摄影风里，人脸和光线比较自然；
- 国潮插画里，红金配色、卷轴、祥云、文房四宝等元素符合预期。

## 5. 视频模型：Agnes-Video-2.0

原文测试了两组视频：

1. 漫剧风格龙舟比赛；
2. 卡通小狐狸学编程。

## 6. 视频能力测试一：龙舟比赛短视频

### 场景要求

- 漫剧风格；
- 15 秒；
- 端午节江面；
- 十几条龙舟冲出起点；
- 鼓手、桨手、水花、冲线庆祝；
- 远景、中景、特写切换；
- 适度镜头晃动；
- 暖色日光；
- 不要文字和水印。

### 原文观察

作者认为视频节奏较稳：

- 起点冲刺、水花和光感明显；
- 鼓手、桨手口型和节拍匹配；
- 冲线庆祝有叙事闭环；
- 轻微镜头晃动增强真实感。

## 7. 视频能力测试二：卡通小狐狸学编程

### 场景要求

- 卡通动画风格；
- 10 秒；
- 小狐狸在森林树桩前写代码；
- 有思考、敲键盘、屏幕出现绿色代码、成功庆祝；
- 松鼠和兔子围观；
- 温暖光线；
- 不要文字和水印。

### 原文观察

作者认为多角色、动作衔接和氛围细节较稳定，没有明显“换动作就换脸”的问题。

## 8. 多模态使用建议

- 图像生成适合多跑几版，从中挑选；
- 架构图类 prompt 要明确层级、组件、连线、方向、配色；
- 视频 prompt 要明确镜头顺序、动作、时长、风格和不要什么；
- 免费模型适合高频试错，但不应默认一次出片即投产。
