# 语言面试与题库资源

## 一句话结论

语言题库是后端面试复习的专项入口，应该按目标岗位选择 Java、Go、C++ 或测试开发路线，而不是一次性平均用力。

## Java/后端

- Java 面试题适合覆盖 Java 基础、并发、JVM、Spring 和常见中间件。
- 大厂后端面经适合作为真实面试题库，用来模拟追问。

## Go

- Go 面试题正文量最大，覆盖基础、Channel、Slice、Map、sync、GMP、GC、内存管理等。
- 当前 vault 已有 Go 知识库，可与 [[04 知识资料/编程语言/Go/00 Go 知识库索引|Go 知识库索引]] 互补。

## C++

- C++ 题库覆盖基础、面向对象、STL、智能指针、现代 C++、问题排查等。
- 适合 C++ 后端、基础架构、客户端和系统开发方向。

## 测试开发

- 测试开发题库覆盖测试理论、业务测试、自动化测试和性能测试。
- 适合测试开发、质量工程和自动化测试方向。
