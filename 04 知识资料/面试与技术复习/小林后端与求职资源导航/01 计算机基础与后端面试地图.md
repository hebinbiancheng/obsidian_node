# 计算机基础与后端面试地图

## 一句话结论

小林 Coding 的后端知识体系适合作为“计算机基础 + 后端面试”的总入口：网络、操作系统、MySQL、Redis 是基础四件套，语言面试和项目面经是求职落地点。

## 基础四件套

- 图解网络：HTTP/HTTPS、TCP/UDP、IP、连接管理、网络排障等。
- 图解系统：进程、线程、内存管理、文件系统、网络系统等。
- 图解 MySQL：索引、事务、MVCC、锁、日志、Buffer Pool、执行计划等。
- 图解 Redis：数据结构、持久化、缓存淘汰、高可用、缓存一致性等。

## 后端面试入口

- Java 面试题：Java 基础、并发、JVM、Spring、MySQL、Redis、消息队列。
- 大厂后端面经：按公司和岗位聚合真实面试题，适合做查漏补缺。
- Go/C++/测试开发题库可作为语言方向专项复习。

## 复习建议

1. 先用基础四件套建立概念地图。
2. 再选择目标语言题库做专项补强。
3. 最后用大厂面经校验真实追问深度。
