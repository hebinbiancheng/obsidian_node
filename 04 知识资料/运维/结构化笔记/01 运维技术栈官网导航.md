## 常见问题

### Q1：学习新组件时，应该先看官方文档还是二手教程？

**A：** 优先看官方文档。二手教程适合快速入门（30 分钟内跑通），但排查复杂问题时，官方文档、版本 Release Notes 和 GitHub Issues 更可靠。尤其是 Kubernetes、Prometheus、Terraform 等版本差异大的工具，二手教程可能已过时。

### Q2：国内访问部分官网慢怎么办？

**A：** 优先使用国内镜像站（清华、阿里云、华为云），Kubernetes 组件可使用阿里云容器镜像服务（registry.aliyuncs.com/google_containers），Docker Hub 可使用国内代理加速。
