# LibTV：AI 视频 Agent + Skill Hub

> 原文：[微信文章](https://mp.weixin.qq.com/s/--vrFFyZmr4qj_XttfOhfA) · 2026-07-14
> 网站：www.liblib.tv

---

## 一句话总结

从「搭节点写提示词」升级到「说话就行」——LibTV Agent 理解最终目标、自动规划视频制作全流程（创意→剧本→角色→分镜→生成→剪辑→配音→配乐→字幕），内置近 100 个专业 Skill，支持自定义 Skill 沉淀。

---

## 核心能力

| 能力 | 说明 |
|------|------|
| **对话式创作** | 只说需求 + 上传素材 → Agent 自动规划节点工作流 |
| **全流程覆盖** | 策划 → 剧本 → 角色 → 分镜 → 生成 → 剪辑 → 配音 → 配乐 → 字幕 |
| **灵活修改** | 对话中直接改，或手动调节点 |

---

## Skill Hub（近百个专业 Skill）

由广告导演、MV 导演、短剧团队、品牌创意、自媒体创作者等沉淀的真实项目经验，可直接调用。

| Skill 示例 | 效果 |
|-----------|------|
| 一键爽感轰炸流汽车 TVC | 产品多视图 + 场景分镜，节奏专业 |
| 皮克斯动画广告 | 带剧情短片的完整制作 |
| 新中式美学 TVC | 国风质感产品短片，概念图 + 模特图 |

**Skill 价值**：从「写提示词」到「直接复用」，把已验证的专业流程变成一键工具。

---

## 自定义 Skill 沉淀

1. 前期探索阶段：理解客户业务 → 提炼品牌理念 → 校准文案风格 → 测试画面形式 → 梳理环节衔接
2. 把这些内容「蒸馏」为 Skill 文件包
3. 上传到 LibTV → 直接调用

**案例**：亲子游短视频 Agent——输入主题 → 自动完成文案、分镜、视频生成、配音、配乐。

---

## 关键启示

> 技术会降低执行门槛，但知道业务真正需要什么、能拆解模糊需求为清晰流程、并沉淀为可复用的 Skill——这依然很难被替代。

未来拉开差距的不是谁更会死磕提示词，而是谁更懂业务、更会梳理流程、更有能力把经验变成 Skill。
