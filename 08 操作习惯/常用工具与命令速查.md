## Hermes CLI

| 用途 | 命令 |
|------|------|
| Gateway 状态 | `hermes gateway status` |
| Gateway 重启 | `hermes gateway restart`（必须在外部终端执行） |
| Gateway 配置 | `hermes gateway setup` |
| 安装技能包 | `hermes skills install <name>` |
| 搜索技能 | `hermes skills search <name>` |
| 查看技能列表 | `hermes skills list` |
| 查看配置 | `hermes config show` |
| 修改配置 | `hermes config set <key> <value>` |

## Git 操作

| 用途 | 命令 |
|------|------|
| 查看最近改动 | `git diff HEAD~1` |
| 查看状态 | `git status` |
| 二分定位 bug | `git bisect start / bad / good` |
| 查看提交历史 | `git log --oneline -10` |

---

## 文件操作（用 Hermes 工具，不用 shell）

| 用途 | Hermes 工具 | ❌ 避免使用 |
|------|------------|------------|
| 读文件 | `read_file` | `cat` / `head` / `tail` |
| 写文件 | `write_file` | `echo >` / `cat << EOF` |
| 编辑文件 | `patch` | `sed` / `awk` |
| 搜索内容 | `search_files(target='content')` | `grep` / `rg` / `find` |
| 列表目录 | `search_files(target='files')` | `ls` / `dir` |
